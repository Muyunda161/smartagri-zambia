# 🌱 SmartAgri Zambia

AI-powered smart agriculture app for Zambian farmers

SmartAgri Zambia is a mobile application concept that uses AI and weather data to help small-scale and commercial farmers in Zambia make smarter farming decisions — covering localized weather forecasting, planting timing, irrigation, fertilizer scheduling, and pest/climate risk alerts.

## 📌 Project Status
🚧 Prototype stage — building toward a full Android (Java) application with a Java Spring Boot backend.

## 🎯 Core Goal
To support small-scale and commercial farmers in Zambia by using mobile technology and AI to provide:
- Accurate, localized weather forecasting
- Smarter farming decisions
- Improved crop productivity and climate resilience

See [`docs/project-overview.md`](docs/project-overview.md) for the full project breakdown.

## 🤖 Features (Prototype)
- 🏡 Dashboard — weather widget, farm stats, daily alerts
- 🌤️ Weather — 7-day forecast, rain probability, storm alerts
- 🌾 Crops — crop info, water needs, farming calendar
- 📋 Recommendations — AI-prioritized action items
- 🤖 AI Farm Advisor — live chat powered by the Claude API, tuned for Zambian farming context

## 🛠️ Tech Stack
Current prototype: React (JSX), CSS, Claude API
Planned production build: Java (Android Studio), Java Spring Boot, Claude API

## 🇿🇲 Local Context
Built with Zambian conditions in mind — supports maize, groundnuts, beans, sorghum, cassava, and sunflower, with a roadmap toward rice. Designed for low-data environments and varying levels of digital literacy.
